HOW IT WORKS

GLP-1の3つの働き

GLP-1受容体作動薬は、体の自然なメカニズムにアプローチしてダイエットをサポートします。

🧠
食欲を抑える

脳の満腹中枢に直接作用し、食欲を自然に抑制。食べ過ぎを防ぎ、無理なくカロリーコントロールが可能になります。

📉
血糖値を下げる

インスリン分泌を促進し、食後の血糖値上昇を緩やかにします。血糖値の安定が、脂肪の蓄積を抑えることにつながります。

🔥
胃の動きを穏やかに

胃の動きを穏やかにし、食事の消化吸収を緩やかにします。少量の食事でも満足感が持続し、自然なカロリー制限につながります。

ご注意事項
  • 本治療は自由診療（保険適用外）です。費用は全額自己負担となります。
  • 効果には個人差があります。すべての方に同様の効果が保証されるものではありません。
  • GLP-1受容体作動薬の副作用として吐き気・嘔吐・下痢・便秘・腹痛・食欲減退などの消化器症状が生じる場合があります。多くは使用初期に出やすく、時間とともに軽減します。
  • まれに低血糖（冷汗・動悸・手足の震え・めまい）、急性膵炎（激しい上腹部痛・背部痛・嘔吐）、胆嚢炎などの重篤な副作用が報告されています。異常を感じた場合は直ちに医師にご相談ください。
  • SGLT2阻害薬（フォシーガ・ルセフィ）では、尿路感染症・性器感染症・頻尿・口渇・脱水などが生じる場合があります。十分な水分補給を心がけてください。
  • 医師のオンライン診察が必要です。問診内容によっては処方できない場合があります。
  • 妊娠中・授乳中の方、膵炎の既往がある方、甲状腺髄様がん・MEN2の既往または家族歴がある方、重度の腎機能障害・透析中の方、1型糖尿病の方は処方できません。
  • 医療広告ガイドラインに基づく表記もご確認ください。
FAQ

よくある質問

GLP-1ダイエットに関するよくある疑問にお答えします。

Q
効果はいつから出ますか？
+
個人差がありますが、使用開始から2〜4週間で食欲の変化を感じ始める方が多いです。体重への効果は1〜3ヶ月ほどで現れることが多く、継続使用によりさらなる効果が期待できます。
Q
副作用はありますか？
+
GLP-1受容体作動薬では、主な副作用として吐き気・下痢・便秘・腹部不快感が報告されています。これらは使用初期に出やすく、多くの場合は時間とともに軽減します。まれに低血糖や急性膵炎の報告があります。SGLT2阻害薬では頻尿・口渇・尿路感染症などが生じることがあります。気になる症状があればLINEでご相談ください。
Q
他の薬との併用はできますか？
+
現在服用中のお薬がある場合は、オンライン診察時に必ずお伝えください。医師が安全性を確認した上で処方の可否を判断します。特に糖尿病治療薬を使用されている方は必ずお知らせください。
Q
解約・休止はできますか？
+
はい、縛りは一切ありません。次回発送日の3日前までにLINEでご連絡いただければ、いつでも解約・休止が可能です。違約金や解約手数料も発生しません。

Q
いつから効果が出ますか？
+
肌の生まれ変わり（ターンオーバー）には約28〜70日かかるため、効果を実感するには最低1〜3ヶ月の継続が目安です。美白効果は3〜6ヶ月の継続をおすすめします。定期便なら飲み忘れを防ぎ、継続しやすくなります。
Q
市販のサプリと何が違いますか？
+
医師が処方する内服薬は、市販サプリよりも成分の含有量・品質基準が高く、医薬品として効果が認められています。また、お肌の状態や生活習慣に合わせて医師が最適な処方を選択するため、より効果的なケアが可能です。
Q
プランは選べますか？
+
はい、お悩みや目的に合わせて既存プランからお選びいただけます。「シミが気になる」「肌荒れを改善したい」など、お悩みをLINEでご相談いただければ、最適なプランをご提案します。
Q
副作用はありますか？
+
いずれも比較的安全性の高い医薬品ですが、胃部不快感・悪心・下痢などの消化器症状が現れることがあります。症状が現れた場合は服用を中止し、医師にご相談ください。トラネキサム酸は血栓リスクがあるため、ピル服用中の方・血栓症の既往がある方は服用できません。

医療広告ガイドラインについて

キレイレポオンラインは、厚生労働省が定める「医業若しくは歯科医業又は病院若しくは診療所に関する広告等に関する指針（医療広告ガイドライン）」に基づき、適切な情報提供を行うことを基本方針としています。

1. 医療広告ガイドラインとは

医療広告ガイドラインとは、医療機関が行う広告について、虚偽・誇大な表現や患者の判断を誤らせる表現を禁止し、患者が適切な医療機関を選択できるよう、医療広告の内容を規制するために厚生労働省が策定した指針です。

当サービスはオンライン診療プラットフォームとして、提携医療機関を通じて適法に診療・処方を行っており、本指針を遵守した情報発信に努めています。

2. 当サービスの広告における基本方針

  1. 虚偽・誇大表現の禁止：科学的根拠のない効果・効能の表現、実際と異なる内容の表記は一切行いません。
  2. 比較優良広告の禁止：他の医療機関や他社サービスとの比較により、優位性を不当に強調する表現は行いません。
  3. 費用の明確な表示：診察料・処方料・送料等の費用は明確に表示し、隠れた費用が生じないよう努めます。
  4. 限定解除要件への対応：当サービスのウェブサイトは、医療広告ガイドラインにおける「限定解除」の要件（問い合わせ先の明示、自由診療の費用明示、治療内容・リスクの明示等）を満たした上で情報を掲載しています。

3. 体験談・口コミ等の取り扱い

医療広告ガイドラインでは、患者の主観的な体験談を広告として掲載することは原則として禁止されています。当サービスでは、個人の感想・体験談については、その旨を明示した上で掲載し、「すべての方に同様の効果が得られるものではない」という注記を必ず付記します。

4. 処方薬・治療に関する情報の表示

  1. GLP-1受容体作動薬（セマグルチド・リラグルチドほか）、メトホルミン等の処方薬については、国内承認の有無・適応外使用の旨を明記しています。
  2. 効果・効能の表現は、医学的・科学的根拠に基づいた範囲に留め、過度な期待を抱かせる表現は使用しません。
  3. 副作用・注意事項については、各診療ページおよび診察時に医師より説明いたします。

5. 自由診療であることの明示

当サービスで提供する診療・処方は、すべて健康保険の適用外となる自由診療です。費用は全額自己負担となります。各プランの料金は、サービスページに明示しています。
